# Compiler Design - Mid Evaluation (LEX Programs)

This repository contains Lex (Flex) programs developed for the Compiler Design mid evaluation.

## Folder: Mid-Evaluation-LEX

It includes the following programs:

- Word Count Program
- Token Identifier
- Infix to Postfix Conversion
- Comment Detection
- Keyword Checker
- Identifier Validator

## How to Run

lex filename.l  
gcc lex.yy.c -o output -ll  
./output

## Purpose

These programs demonstrate basic lexical analysis concepts using Lex/Flex.
